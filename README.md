# FAST FEST 2026 Event Portal

FAST FEST 2026 Event Portal is a simple and responsive front-end web project developed using HTML and CSS. The portal is designed to provide event information, schedules, registration details, and highlights for the FAST FEST 2026 university event.

## 🚀 Features
- Responsive homepage design
- Event information section
- Event schedule display
- Registration section
- Navigation bar
- Attractive UI layout
- Simple and clean styling
- Mobile-friendly structure

## 🛠️ Technologies Used
- HTML5
- CSS3
